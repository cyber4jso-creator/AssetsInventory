# Project Roadmap

Official delivery plan for the Asset Inventory Management System.

---

## Sprint 1 — Identity, Data Consistency, RBAC, Frontend Stabilization

**Status:** Complete (pending final approval)

- Demo users (single source of truth)
- Mock data consistency across modules
- Role-based access control (RBAC)
- Frontend stabilization (dashboard, notifications, profile, forms)
- Data scope contract prepared for backend (`auth/scope`)

---

## Sprint 2 — Frontend Polish & UX

**Status:** Not started

- UX improvements
- Forms refinement
- QR layout
- Print layout
- Report layout
- Component consistency
- Empty states
- Loading states
- Confirmation dialogs
- Toast notifications

> Backend work is **not** part of Sprint 2.

---

## Sprint 3 — Backend Development

**Status:** Not started

- NestJS
- Prisma
- PostgreSQL
- Authentication integration
- REST APIs
- Frontend integration (replace mock data with scoped API calls)
